4  DataFrame Querying

In the previous chapter we discussed accessing rows of a DataFrame by row position. In practice, I donâ€™t find this particularly useful. Rather, I find that I usually access rows by some kind of logical condition, which is referred to as querying the DataFrame.

In this chapter we discuss two ways of querying a DataFrame:

  1. masking
  2. the DataFrame.query() method.

4.3 Comparison and DataFrame Columns

As discussed in a previous chapter, a column of a DataFrame is a Series object, which is a souped up numpy.array (think vector or matrix).

Letâ€™s separate out the adjusted column of df_spy and assign it to a variable.

Recall that a pandas.Series is smart with respect to component-wise arithmetic operations, meaning it behaves like a vector from linear algebra. This means that arithmetic operations are broadcasted as you might expect.

For example, division by 100 is broadcasted component-wise.

ser_adjusted / 100
0     4.1528
1     4.1758
2     4.2077
       ...  
19    4.2573
20    4.2749
21    4.2542
Name: adj_close, Length: 22, dtype: float64

It is a convenient fact that this broadcasting behavior also occurs with comparison, and produces a Series of booleans.

The following code checks which elements of ser_adjusted are greater than 425.

ser_test = (ser_adjusted > 425)
ser_test

1     False
2     False
      ...  
19     True
20     True
21     True
Name: adj_close, Length: 22, dtype: bool

Letâ€™s check that the resulting variable ser_test is a pandas.Series.

type(ser_test)
pandas.core.series.Series

And finally letâ€™s observe the .values elements of ser_test.

print(ser_test.values)
[False False False False False False False False False False False False
 False False False False  True  True  True  True  True  True]

A few observation about what just happened:

  1. When we compare a Series of numerical values (ser_adjusted) to a single number (425), we get back a Series of booleans (ser_test).

  2. We have that ser_test[i] = (ser_adjusted[i] > 425).

  3. So the comparison operation was broadcasted as advertised.

Letâ€™s modify lst_bool once again, by setting the 1st entry of df_spy to True, and then once again feed it into df_spy.

Punchline: What is returned by the code df_spy[lst_bool] will be a DataFrame consisting of all the rows corresponding to the True entries of lst_bool.
